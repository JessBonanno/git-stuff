# git-stuff

This repo will be used to go over some very basic git commands and demonstrate a standard git flow

### Things we will cover:
* Create a git repo
* Clone a git repo
* Fork a git repo

* Checkout new branch
* Checkout existing branch

* Git add (file and all)
* Git commit (-m & without)
* Git push
* Git lens

* Pull Requests
* Merges
* Git pull
* Git fetch

* Merge conflicts
* Git rebase
* Git rebase —abort
* Git rebase —continue
